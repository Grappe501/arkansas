# Purpose

The success of Citizens United Facts should never depend on one individual.

The institution must be designed so that knowledge, leadership, relationships, research, and operational practices survive leadership transitions and continue improving over time.

Institutional continuity is a deliberate design choice.

Knowledge · Processes · Relationships · Culture · Standards · Institutional memory
