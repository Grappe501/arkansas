# Purpose

No single organization can educate an entire state alone.

Citizens United Facts is designed to become the **connective infrastructure** that allows organizations across Arkansas to cooperate while maintaining their own identity, leadership, and mission.

The Coalition Network does not control partner organizations. Its purpose is to help them collaborate more effectively around civic education.

---

# Coalition Philosophy

Every partner brings unique strengths. Citizens United Facts provides:

Shared knowledge · Shared tools · Shared calendars · Shared educational resources · Shared communication · Shared opportunities

Greater collective impact without sacrificing organizational independence.

---

# Coalition Principles

Every partnership should be built on:

Mutual respect · Transparency · Voluntary participation · Shared educational goals · Evidence-based information · Local leadership · Long-term trust

The platform should strengthen existing organizations rather than replace them.
