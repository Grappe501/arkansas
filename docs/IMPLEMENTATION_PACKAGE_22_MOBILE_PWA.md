# Purpose

Most volunteers and citizens will interact with Citizens United Facts from a phone.

The mobile platform is not a companion application. **It is a first-class experience.**

Every feature available on the desktop platform should be available on mobile unless there is a compelling usability reason otherwise.

---

# Mobile Philosophy

Design for the volunteer standing in a library. Design for the Education Leader walking into a community meeting. Design for the citizen attending a city council meeting. Design for someone with one free minute waiting in line.

Mobile should support real life.

---

# Progressive Web App (PWA)

Version 1 should be installable as a Progressive Web App.

Benefits include: Home screen installation · Fast startup · Offline capabilities · Push notifications · Background synchronization · App-like experience without requiring an app store

Native iOS and Android applications can be developed later if needed.
