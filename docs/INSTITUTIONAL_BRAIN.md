# Purpose

By Build #40, the platform is no longer simply a website.

It has become a structured body of knowledge.

This build defines the Institutional Brain—the system that understands how every person, event, court case, law, constitutional principle, dataset, article, educational resource, organization, and Arkansas county relates to every other part of the platform.

Everything becomes connected.

Nothing stands alone.

Mission Control should visualize this Institutional Brain as the platform grows.

---

# Vision

Imagine walking into the world's largest room dedicated to understanding *Citizens United*.

Instead of shelves filled with books, every idea is connected by visible threads.

Touch one idea and you immediately see:

* What came before it.
* What followed it.
* Which court cases influenced it.
* Which laws relate to it.
* Which data supports it.
* Which articles explain it.
* Which Arkansas educational resources reference it.
* Which community presentations use it.
* Which coalition partners teach it.

That is the Institutional Brain.

# Governing Principle

The Citizens United Facts platform should never become a collection of disconnected pages.

It should become a living civic knowledge institution where every fact, every source, every lesson, every county, every educator, every organization, and every research document strengthens every other part of the system.

When the Institutional Brain is complete, the platform will not simply contain information.

It will understand how all of that information fits together, making it easier for every Arkansan to learn, explore, verify, and teach one of the most significant Supreme Court decisions in modern American history.
