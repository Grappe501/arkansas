# Purpose

**Build #90** establishes the complete guide that allows future volunteers and leaders to understand, operate, improve, and preserve Citizens United Facts without relying on institutional memory held by a few individuals.

If every founding volunteer stepped away tomorrow, a new generation should continue the mission using this manual and Mission Control.
