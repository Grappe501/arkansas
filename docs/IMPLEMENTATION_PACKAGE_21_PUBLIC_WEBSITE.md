# Purpose

The public website is the **front door** to Citizens United Facts.

Most Arkansans will form an opinion of the institution within the first minute of visiting. The experience should communicate:

Trust · Competence · Clarity · Curiosity · Hope · Service

The website should invite people to explore rather than overwhelm them.

---

# Core Philosophy

Every page should answer three questions:

- **Why should I care?**
- **What can I learn?**
- **What can I do next?**

If a page cannot answer all three, it is incomplete.
