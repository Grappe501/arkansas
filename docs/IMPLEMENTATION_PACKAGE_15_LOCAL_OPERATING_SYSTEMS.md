# Purpose

The statewide institution is built locally.

Every county, city, and neighborhood should have its own operating system while remaining fully integrated into the statewide Citizens United Facts platform.

Mission Control coordinates statewide. County Operating Systems coordinate locally. One institution with distributed leadership.

---

# Local-First Philosophy

```text
Arkansas → County → City → Neighborhood → Community Conversation → Citizen
```

Every level strengthens the level above it.

# Governing Principle

A statewide institution is not built from a central office. It is built through thousands of local relationships working from one shared mission.

When every county is organized, every city is connected, every neighborhood has a pathway to participate, and every citizen can easily find trusted civic education, Arkansas gains a permanent infrastructure for informed self-government.
