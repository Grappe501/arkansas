# Purpose

Automation should eliminate repetitive administrative work — **not human relationships**.

The **Automation Engine** coordinates routine institutional processes so volunteers can spend their time researching, teaching, mentoring, organizing, and building communities.

Automation supports people. People lead the mission.

---

# Automation Philosophy

Before automating a process, ask:

- Is it repetitive?
- Is it rules-based?
- Does automation improve consistency?
- Does it reduce volunteer workload?
- Does it preserve human oversight where appropriate?

Only then should it be automated.
